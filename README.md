# 📘 Quizi 

## 🚀 Project Description  

### English  
This project is a private web-based platform designed for creating and managing online assessments.  
It is currently in its **initial demo version**, focusing on core functionality, while advanced features are planned for upcoming releases.  

The application follows a layered architecture with clear separation of concerns, including:  

- **Interface & Repository layers** for data access and abstraction  
- **Tools & ViewModels** to support business logic  
- **Fluent mapping with EF Core (Code First)** for professional database design  
- **Razor Pages architecture** with extendable structure  
- Planned support for **RESTful APIs & Minimal APIs** for modular growth  
- Advanced **encryption and security mechanisms** (under development)  

#### 🔮 Future Features  
- **Video interview integration** for interactive assessments  
- **Professional UI/UX with modern design patterns**  
- **Expanded management dashboards and reporting tools**

### فارسی  
این پروژه یک پلتفرم تحت وب خصوصی برای طراحی و مدیریت آزمون‌های آنلاین است.  
در حال حاضر در **نسخه‌ی دمو اولیه** قرار دارد که روی قابلیت‌های اصلی تمرکز دارد، و در نسخه‌های آینده امکانات پیشرفته اضافه خواهند شد.  

معماری پروژه به صورت لایه‌ای طراحی شده و شامل بخش‌های زیر است:  

- لایه‌های **Interface و Repository** برای مدیریت داده و انتزاع دسترسی  
- **ابزارها و ViewModelها** جهت پیاده‌سازی منطق تجاری  
- **Fluent Mapping در EF Core (Code First)** برای طراحی حرفه‌ای پایگاه داده  
- استفاده از الگوی **Razor Pages** با ساختار قابل توسعه  
- قابلیت توسعه جهت افزودن **RESTful API و Minimal API** برای ماژولار شدن سیستم  
- پیاده‌سازی **رمزنگاری و امنیت پیشرفته** (در دست توسعه)  

#### 🔮 قابلیت‌های آتی  
- **مصاحبه تصویری** برای آزمون‌های تعاملی  
- **رابط کاربری حرفه‌ای و جذاب** با الگوهای طراحی مدرن  
- **داشبوردها و گزارش‌های مدیریتی پیشرفته**
