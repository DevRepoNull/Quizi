English
A private web-based platform for creating and managing online exams. Supports user, role, question, and form management. Current version is demo; future updates include video interviews and a professional UI.

فارسی
پلتفرم تحت وب خصوصی برای طراحی و مدیریت آزمون آنلاین. مدیریت کاربر، نقش، سوال و فرم. نسخه فعلی دمو است؛ در آینده مصاحبه تصویری و رابط کاربری حرفه‌ای اضافه می‌شود.
